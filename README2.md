# 🎎 Anime IPTV

Catálogo automatizado de animes com playlist IPTV e site publicado via GitHub Pages.

## 🌐 Acesso
- Site: https://kaykewf13.github.io/anime-iptv/
- Playlist M3U: https://raw.githubusercontent.com/kaykewf13/anime-iptv/main/output/anime_channels.m3u

## 📂 Funcionalidades
- Catálogo automático de animes
- Separação por categorias
- Playlist IPTV com fontes públicas
- Atualização automática via GitHub Actions
- Interface web com GitHub Pages

## 🚀 Atualização
O projeto é atualizado automaticamente a cada 12 horas.
